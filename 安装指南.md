# 安装依赖指南

## 步骤 1: 安装 Node.js 和 npm

你的系统还没有安装 Node.js，这是运行这个项目的前提。请按照以下步骤安装：

### 方法一：从官网下载安装（推荐）

1. **访问 Node.js 官网**：
   - 打开浏览器，访问：https://nodejs.org/
   - 或者直接访问中文镜像：https://nodejs.org/zh-cn/

2. **下载 LTS 版本（长期支持版）**：
   - 点击绿色的 "推荐给大多数用户" 按钮
   - 这会下载 Windows 安装程序（.msi 文件）
   - 目前推荐版本是 v20.x 或 v18.x

3. **运行安装程序**：
   - 双击下载的 .msi 文件
   - 按照安装向导的提示进行操作
   - **重要**：确保勾选 "Add to PATH" 选项（默认会勾选）
   - 点击 "Next" 完成安装

4. **验证安装**：
   - 关闭当前的命令行窗口
   - 重新打开 PowerShell 或命令提示符
   - 运行以下命令验证：
   ```bash
   node --version
   npm --version
   ```
   - 如果显示版本号（例如 v20.10.0 和 10.2.3），说明安装成功！

### 方法二：使用包管理器安装（高级用户）

如果你已经安装了 Chocolatey 或 Scoop：

**使用 Chocolatey：**
```bash
choco install nodejs
```

**使用 Scoop：**
```bash
scoop install nodejs
```

## 步骤 2: 安装项目依赖

安装完 Node.js 后，在项目目录中运行：

```bash
npm install
```

这个命令会：
- 读取 `package.json` 文件
- 下载所有必需的依赖包
- 将它们安装到 `node_modules` 文件夹中

**注意**：首次安装可能需要几分钟时间，取决于你的网络速度。

## 步骤 3: 启动开发服务器

依赖安装完成后，运行：

```bash
npm run dev
```

然后打开浏览器访问：`http://localhost:5173`

## 常见问题

### 问题 1: 安装后仍然提示找不到 npm
**解决方案**：
- 关闭所有命令行窗口
- 重新打开 PowerShell 或命令提示符
- 如果还是不行，重启电脑

### 问题 2: npm install 很慢或失败
**解决方案**：
- 使用国内镜像源（淘宝镜像）：
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```
- 然后再运行 `npm install`

### 问题 3: 权限错误
**解决方案**：
- 以管理员身份运行 PowerShell
- 或者在项目目录右键，选择 "在终端中打开"

## 需要帮助？

如果遇到任何问题，可以：
1. 检查 Node.js 是否正确安装：`node --version`
2. 检查 npm 是否正确安装：`npm --version`
3. 查看错误信息并搜索解决方案

安装完成后，你就可以开始使用这个区块链交易数据面板了！🎉
