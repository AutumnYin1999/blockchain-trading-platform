# 如何找到 GitHub 仓库地址

## 方法 1: 在 GitHub Desktop 中查看

1. 打开 GitHub Desktop
2. 点击顶部菜单栏的 **"Repository"** → **"Repository Settings"**
3. 在设置窗口中，找到 **"Remote repository"** 部分
4. 你会看到类似这样的地址：
   ```
   https://github.com/你的用户名/你的仓库名.git
   ```
   或者
   ```
   git@github.com:你的用户名/你的仓库名.git
   ```

## 方法 2: 在 GitHub Desktop 中查看（更简单）

1. 打开 GitHub Desktop
2. 点击顶部菜单栏的 **"Repository"** → **"View on GitHub"**
3. 这会直接在浏览器中打开你的 GitHub 仓库页面
4. 在浏览器地址栏中，你会看到类似：
   ```
   https://github.com/你的用户名/你的仓库名
   ```

## 方法 3: 在 GitHub 网页上查看

1. 登录 GitHub 网站
2. 点击右上角的头像
3. 点击 **"Your repositories"**
4. 找到你的项目，点击进入
5. 在仓库页面，点击绿色的 **"Code"** 按钮
6. 你会看到仓库地址（HTTPS 或 SSH）
